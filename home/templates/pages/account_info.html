API Usage Policy

The following guidelines are designed to ensure the security and proper use of your API token:

  • Login: Users must log in to obtain a token. The token is a unique identifier that allows you to access various services securely. Ensure that your credentials are kept private and secure to avoid unauthorized access.
  • Security: Do not share your token with anyone, including friends or family. Your token acts like a password, granting access to your account and data. If you believe your token has been compromised, please regenerate it immediately.
  • Expiration: The token will expire after 30 minutes of inactivity. After expiration, you will need to log in again to obtain a new token. Regularly check for token validity to ensure uninterrupted access to services.
  • Access Rights: The token is only valid for legitimate API requests. Using the token for unauthorized actions or requests may result in suspension or termination of your account. Make sure to review the API documentation for appropriate usage.
  • Rate Limiting: To maintain the quality of service, API requests are subject to rate limits. Exceeding these limits may result in temporary suspension of access. Please refer to the API documentation for details on rate limits and best practices for efficient usage.
  • Data Privacy: Any personal data accessed through the API is subject to our privacy policy. Ensure that you comply with data protection regulations and do not misuse any information obtained via the API.
  • Support Contact: If you encounter issues or have questions regarding the API, please contact support via email. Our support team is available to assist you with any queries or concerns you may have.
